# 🕊️ Grateful — 12 Step Recovery Companion

A beautiful, mobile-first web app to support recovery through the 12 Steps, daily gratitude journaling, affirmations, and prayer.

## Features

- 📋 **12 Step Tracker** — tap each step to read it and mark it complete
- 📖 **Gratitude Journal** — log 3 things you're grateful for each day
- 🔥 **Streak Counter** — tracks your daily journaling streak
- 💬 **Daily Affirmations** — rotating recovery affirmations
- 🙏 **Prayers** — Serenity Prayer, Third Step Prayer, Seventh Step Prayer
- 😌 **Mood Check-in** — log how you're feeling each day
- 💾 **Local Storage** — your data persists between sessions

## Getting Started

No installation needed. Just open the file in any browser:

```bash
git clone https://github.com/11kinglsat-source/grateful-app.git
cd grateful-app
open index.html
```

Or visit the live demo via GitHub Pages.

## Tech Stack
- Vanilla HTML, CSS, JavaScript
- Google Fonts (Playfair Display + DM Sans)
- LocalStorage for data persistence
- Zero dependencies
